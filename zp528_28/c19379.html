ビジネスバックパック 拡張タイプ ブラック
■大容量で十分な収納力を備えた拡張機能付きのビジネスバックパックです。
■背負うことで両手が空くため、雨の日や自転車移動などの際も快適に動けます。
■拡張前は合計約27Lの荷物を収納でき、拡張後は合計約31Lの荷物を収納可能なので出先で荷物が増えても安心です。
■8cm拡張可能なトランク気室には、荷物の偏りを防ぐクロスベルトが付いています。
■トランク気室は、キャリーケースのように180度フルオープンで開くため着替えなどの荷物が取り出しやすいです。
■15.6インチのパソコンなどが収納できる内装ポケットには、端末を優しく保護するフリース素材を採用。移動中の揺れによる中の荷物との衝突を防ぎます。
■本体前面には上部と下部にフロントポケットがあり、下部ポケットは仕切りポケットが内蔵されています。出し入れ頻度の高い小物類の収納に便利です。
■本体背面にファスナー開閉式の隠しポケットを装備しています。背負いながらでも財布やスマートフォンなどをスムーズに取り出せます。
■本体両側面には、折りたたみ傘やペットボトルなど収納できるサイドポケットが付いています。背負ったままの出し入れもしやすく便利です。
■肩・背中に触れる部分は、通気性の良いエアーメッシュ素材を採用しています。
■チェストベルトによってバッグを体にフィットさせることができ、荷重を分散させることで疲労感を軽減します。
■表面生地には撥水加工を施しています。
■※内部への水の浸入を完全に防ぐものではありません。
■背面にキャリーバーへ固定できるキャリーベルトを配置しています。キャリーケースと一緒に持ち運べるので移動がスムーズになります。
■容量 : 未拡張時 : 約27L
■容量 : 拡張時 : 約31L
■外形寸法 : 未拡張時 : 約W310×約D210×約H470mm ※ハンドル・ショルダーベルト含まず
■外形寸法 : 拡張時 : 約W310×約D290×約H470mm ※ハンドル・ショルダーベルト含まず
■気室数 : 2(メイン気室+トランク気室)
■内寸 : パソコン用オープンポケット : 約W255×約D27×約H365mm ※15.6インチノートパソコンの参考収納寸法
■PC収納部内寸備考 : ※内寸と収納できるパソコンの外寸は異なる場合がございます。詳しくはエレコムホームページの対応表でご確認ください。
■ポケット(前面) : 仕分けスリーブ付きファスナーポケット×1、ファスナーポケット×1
■ポケット(背面) : ファスナーポケット×1
■ポケット(側面) : オープンポケット×2
■ポケット(気室内) : メイン気室 : パソコン用オープンポケット(起毛)×1、タブレット用オープンポケット×1、オープンポケット(メッシュ)×2トランク気室 : ファスナーポケット(メッシュ)×1

■拡張機能 : マチ拡張
■対応モデル : ～15.6インチ ノートパソコン
